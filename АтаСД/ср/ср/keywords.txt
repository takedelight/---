if:умовний оператор для виконання коду за певної умови
else:виконується, якщо умова в операторі if хибна
while:цикл, що виконується доки умова залишається істинною
for:цикл з лічильником для ітерації по діапазону значень
switch:конструкція для вибору одного з багатьох варіантів
int:цілочисельний тип даних (32 біти)
string:послідовність символів Юнікод
bool:логічний тип даних (true або false)
double:число з плаваючою комою подвійної точності
class:основна одиниця для створення об'єктів (ООП)
namespace:контейнер для логічного групування класів
public:модифікатор, що робить член доступним звідусіль
private:модифікатор, що обмежує доступ межами поточного класу
static:вказує, що член належить самому типу, а не об'єкту
void:тип, що вказує на відсутність результату повернення
return:оператор завершення методу та повернення значення
new:оператор для створення нового об'єкта або екземпляра
using:директива для підключення просторів імен
try:блок коду, в якому може виникнути помилка
catch:обробка виняткових ситуацій (помилок) у блоці try
